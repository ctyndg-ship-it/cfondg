# 📊 Dashboard Access

Mở trình duyệt và truy cập:

## http://localhost:3000

---

## Tính năng Dashboard:

| Chức năng | Mô tả |
|-----------|-------|
| 🔄 Crawl Data | Thu thập xu hướng mới |
| ✍️ Generate Scripts | Tạo script từ trends |
| 🚀 Run Full Automation | Chạy cả 2 |
| 📊 Stats Grid | Xem số liệu tổng quan |
| 🔥 Top Trends | Danh sách 10 trends hot nhất |
| 📝 Generated Scripts | Xem script đã tạo |
| ♻️ Auto-refresh | Cập nhật mỗi 30 giây |

---

## Các lệnh khác:

```bash
node index.js dashboard   # Chạy dashboard
node index.js full        # Chạy automation
node index.js export      # Xuất báo cáo
```
